# Enactus FUTO Website

Official website for Enactus FUTO — built with React + Tailwind CSS.

## Tech Stack
- React 18 + Vite
- Tailwind CSS v4
- React Router DOM
- Framer Motion
- React Helmet Async

## Getting Started
npm install
npm run dev

## Folder Structure
src/
  Pages/       → page components
  component/   → reusable components
  assets/      → images and icons
  routers/     → routing config

## Deployment
Hosted on Vercel. Auto-deploys on push to main.
